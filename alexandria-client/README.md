# Alexandria client (Swift)

Kod koji u Alexandria app browseru koristi Eluminatium backend: pretraga kataloga, prikaz aplikacija, preuzimanje.

- `Eluminatium.swift` – UI (tražilica, aurora pozadina, lista appova, samo katalog).
- `EluminatiumService.swift` – API klijent (fetchSearchUI, search, downloadZip, fetchDSL).

Cijeli Alexandria projekt je u drugom repou (TreeOFChoice); ovdje je samo Eluminatium-related kod za pregled.
