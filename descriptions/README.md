# Opisi aplikacija

Ovdje se nalaze JSON datoteke s opisima aplikacija.
Svaka aplikacija može imati vlastitu datoteku `{id}.json`.

Polja:
- `id` – identifikator (mora odgovarati catalog.json)
- `name` – naziv
- `description` – opis
- `keywords` – ključne riječi za pretragu
- `category` – kategorija (opcionalno)
